# 🛍️ Nostra Fashion Website

A responsive fashion e-commerce front-end website built using HTML, CSS, and JavaScript.
The project showcases modern UI components like product filtering, image sliders, smooth scrolling, and responsive navigation.

# 🏠 Home Page

Responsive navigation bar
Promotional offer banner
Image slider for collections
Brand showcase section
Customer experience highlights
New arrivals section
Most wanted products
Limited time offer section
Newsletter subscription
Footer with social media icons

# 👕 Collections Page

Product grid layout
Filter products using:
Occasion (Summer, Party, Beach)
Color (Red, Blue, White, Green)
Arrivals (New / Old)
Real-time product filtering using JavaScript
Search bar for product search
Responsive design

# 📞 Contact Page

Contact form UI
Name, Email, Message fields
Responsive layout
Integrated with main navigation
⚡ Interactive Features

# 🖼️ Image Slider

Users can scroll through featured images using navigation arrows.

# ❤️ Wishlist Toggle

Heart icon changes when users click to mark favorites.

# 📜 Smooth Scrolling

Navbar buttons smoothly scroll to sections:
New Arrival
Most Wanted

# 📱 Mobile Navigation

Responsive side navigation menu for smaller screens.

# 🔎 Product Filtering

Dynamic filtering using checkboxes.
Example filters:
Summer products
Party wear
Color-based products
New arrivals
